# 🔥 Featured Projects

### 🤖 StratoTest — Agentic AI Backtesting Platform
Built a multi-agent AI pipeline capable of converting plain-English trading ideas into executable Python backtests. Implemented self-correcting code generation and an autonomous Optimization Agent to tune strategies for target Sharpe ratios and drawdowns.
*(Python, FastAPI, Streamlit, LLMs)*

### 🎯 TIPE — AI Trade Intelligence Platform
Developed an AI-powered lead ranking platform with vector similarity scoring (Pinecone) and intent prediction. Built a responsive TypeScript frontend with swipe-based approval UI for seamless data-driven trade decisions.
*(HTML, CSS, TypeScript, Python, Pinecone, REST APIs)*

### 🎙️ Speech Correction App
Designed an AI-based speech analysis model for pronunciation correction. Implemented a phoneme-level audio processing pipeline with real-time Firebase-powered updates integrated into a Flutter frontend.
*(Flutter, Python, Firebase)*
